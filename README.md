# Liberta infrastructure and apps

## English

This repo aims to contain all Liberta apps and infrastructure.

apps/liberta-mail-server: this is a first try aiming to deploy a full-capable mail server to run in Liberta infrastructure.
It uses Tofu (Terraform compatible fork) and Ansible.

A noticeable planned feature is the full mail encryption at rest (with the Dovecot's "crypt" plugin).

NOT TO USE IN PRODUCTION
