# Liberta mail server deployment

## English

This repo is a first try aiming to deploy a full-capable mail server to run in Liberta infrastructure.
It uses Tofu (Terraform compatible fork) and Ansible.

A noticeable planned feature is the full mail encryption at rest (with the Dovecot's "crypt" plugin).

NOT TO USE IN PRODUCTION
